How to write a formula

A formula is a list of orders for a turtle holding a pen. It starts pointing up, walks where you tell it, and the magic order T makes it run the whole formula again — smaller. That self-call is the recursion you met in chapter 2.

F1

✏️ Draw a stick

Walk forward drawing a line. The number is the length: 1 = the full current size, 0.5 = half of it.

M0.5

👣 Move without drawing

Same walk, but with the pen lifted — handy to leave gaps.

+60  -45

↪️ Turn

Rotate on the spot: + turns left, − turns right, the number is degrees. The turtle starts pointing up, so +90 faces it left and -90 faces it right.

[ … ]

🌿 Branch

Everything inside brackets is a side trip: the turtle remembers where it was, does the trip, and snaps back to keep going from the same spot.

T0.7

🔁 The self-call

Run the whole formula again from here, at 0.7× the size. This is what makes it a fractal! You can use up to five per formula — the tree uses two, the snowflake three.

# hello

💬 Notes

Anything after a # is ignored — leave yourself notes. Spaces and upper/lower case don't matter either.

Read the tree formula aloud 

F1 [+25 T0.7] [-25 T0.7]

"Draw a stick. Side trip: turn left 25° and do all of this again at 70% size. Snap back. Side trip: turn right 25° and do it all again at 70%." That's the exact rule from chapter 2 — two self-calls, so the sticks double every generation.
